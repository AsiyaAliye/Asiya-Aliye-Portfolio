Implemented a small compiler for a simple language which involved generating an intermediate representation of the code. After the compiler generates the list of instructions, it will execute this list by interpreting it. Meaning the program will traverse the data structure, executing each node by modifying memory locations corresponding to operands. It will then determine and proceed to the next instruction, continuing until there are no more instructions to execute.
